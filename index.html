CoderDojoとは

CoderDojo とはアイルランドから始まったムーブメントです。 プログラミングを使っているボランティアが、小中学生に対して無料でプログラミングを 教えるものです。CoderDojo公式(英語)

CoderDojo Chiba

CoderDojo Chiba は千葉市での活動を中心した CoderDojoです。入門者向けには、MITメディアラボが開発した Scratch を使ってプログラミングの基本を教えていきます。

また、Scratch で物足りない人向けに、 HTML/JavaScript/CSS, PHP など参加者の希望次第で出来る限り教えていきます。

メンター・支援者

CoderDojo Chiba では、千葉でプログラミングを教えたり、支援してくださるボランティアを募集しております。 このページは Github で管理されていますので、賛同していただける方は、自分のプロフィールを追加していただいて Pull Request を送ってください。
