CoderDojo Chiba

CoderDojo Chiba (コーダー道場千葉) は千葉市で開催される小中学生向け無料プログラミング道場です

CoderDojoとは

CoderDojo とはアイルランドから始まったムーブメントです。 ボランティアが、小中学生に対して無料でプログラミングを教える「道場」です。CoderDojo公式(英語)

CoderDojo Chiba

CoderDojo Chiba は千葉市での活動を中心した CoderDojoです。入門者向けには、MITメディアラボが開発した Scratch を使ってプログラミングの基本を教えていきます。

また、Scratch で物足りない人向けに、 HTML/JavaScript/CSS, PHP など参加者の希望次第で出来る限り教えていきます。

千葉県で開催されている CoderDojo は、他にも Kashiwa (柏) もありますので、そちらも合わせて御覧ください。

開催スケジュール

開催は、原則月1度の土日祝です。

開催日程などは、Facebook Group にて告知しておりますので、ぜひご参加ください。

メンター・支援者・支援団体

CoderDojo Chiba では、千葉でプログラミングを教えたり、支援してくださるボランティアを募集しております。 (開催・参加レポートを書いてくれるだけでもOK!) このページは Github で管理されていますので、賛同していただける方は、自分のプロフィールを追加していただいて Pull Request を送ってください。(または、@ooharabucyou に連絡をお願いします。)
